# react_tic_tac_toe

Jogo da Velha (Tic‑Tac‑Toe) em React — projeto para exercício.

## Como usar

1. Se ainda não criou o projeto com `create-react-app`, rode:
   ```bash
   npx create-react-app react_tic_tac_toe
   cd react_tic_tac_toe
   ```
2. Substitua a pasta `src/` e o arquivo `public/index.html` pelo conteúdo deste repositório (ou copie os arquivos manualmente).
3. Instale dependências (se necessário) e rode:
   ```bash
   npm start
   ```
4. O app abrirá em `http://localhost:3000`.
